
```bash

```

---

# ▶️ How to Run

Clone this repository:

```bash
git clone https://github.com/your-username/python_mini_task.git
```

Move into the project folder:

```bash
cd python_mini_task
```

Run any project:

```bash
python file_name.py
```

Example:

```bash
python tic_tac_toe.py
```
